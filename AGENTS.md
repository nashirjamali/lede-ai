# AGENTS.md

## Design Context

Lede is a **product**-register surface (app UI is primary; the `/` landing is a secondary brand surface).

- **`PRODUCT.md`** — strategic context: users, product purpose, brand personality, anti-references, design principles, accessibility (WCAG 2.1 AA). Read it before strategic or UX decisions.
- **`DESIGN.md`** — visual system: tokens, color palette, typography, elevation, components, do's and don'ts. Read it before any UI work so new screens stay on-brand.

**North star:** "The Editorial Desk" — Notion meets The Economist. Ink-on-paper foundation (`#0E1320` ink, `#FAF7F2` cream) with terracotta (`#C24E2E`) as a rare editor's-pencil accent. Newsreader serif display, Albert Sans body, IBM Plex Mono metadata. Editorial calm over dashboard noise; flat by default.

> Note: `SPEC.md` §7.2 describes an "editorial-green" palette, but the shipped code (and `DESIGN.md`) use the ink/cream/terracotta system above. The code and `DESIGN.md` are the source of truth.
